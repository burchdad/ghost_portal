# Trial Operations

Every day, Ghost Portal should answer five questions for Alex:

1. What should I work on?
2. What is due?
3. What is blocked?
4. What requires Stephen's approval?
5. What must I report before ending my shift?

Daily reports allow one report per user per work date. Stephen reviews reports and can request changes.

Approvals are the "Waiting on Stephen" queue and prevent Operations from committing pricing, deadlines, scope, or follow-ups without Founder review.
