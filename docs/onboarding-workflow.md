# Onboarding Workflow

Alex's trial onboarding is stored in `OnboardingModule` records. Progress is stored in `OnboardingCompletion` per user/module.

Initial modules cover company context, Ghost products, Nova, Vega, Echo, GEO, communication standards, confidentiality, security, approval boundaries, the trial plan, and daily-report expectations.

Founder-owned admin editing is modeled and reserved for the onboarding content admin screen.
